របៀបទិញគម្រោងប្រើប្រាស់ KOOMPI Fi-Fi

Updated 29 March 2021

ពេលនេះយើងនឹងណែនាំអ្នកទាំងអស់គ្នាមកចុះឈ្មេាះបង្កើតគណនីនឹងរបៀបទិញគម្រោងដូចខាងក្រោម៖

ជំហានទី១

  • លោកអ្នកត្រូវទាញយកកម្មវិធី KOOMPI Hotspot ពី Google PlayStore សំរាប់អ្នកប្រើប្រាស់ទូរស័ព្ទ Android ។

ជំហានទី២

  • ដំណើរការកម្មវិធី KOOMPI Hotspot ដោយចុចលើនិមិត្តសញ្ញាកម្មវិធី KOOMPI Hotspot នៅលើទូរស័ព្ទ របស់អ្នក (រូបភាពខាងក្រោម)។

ជំហានទី៣

  • អ្នកដែលបានបង្កើតគណនីរូចហើយ គ្រាន់តែបញ្ចូល លេខទូរស័ព្ទ និង ពាក្យសម្ងាត់ អោយបានត្រឹមត្រូវ រូចចុចប៊ូតុង ចូលគណនី ជាការស្រេច។ ចំពោះអ្នកជាសមាជិកថ្មីត្រូវចុចលើពាក្យ បង្កើតគណនី ដើម្បីចុះឈ្មេាះបង្កើតគណនី ថ្មីដោយបំពេញដូចជា៖ លេខទូរស័ព្ទ ពាក្យសម្ងាត់ និង បញ្ចាក់ពាក្យសម្ងាត់ម្ដងទៀត អោយបានត្រឹត្រូវរួចចុចប៊ូតុង បង្កើតគណនី ។

ជំហានទី៤

  • បន្ទាប់មកអ្នកនិងចូលដល់កន្លែងផ្ទៀងផ្ទាត់ លេខទូរស័ព្ទ។ កម្មវិធី KOOMPI Hotspot នឹងផ្ងើរសារទៅលេខទូរស័ព្ទរបស់អ្នកដែល ផ្ទុកលេខកូដចំនួន៦ខ្ទង់។ បន្ទាប់មកអ្នកត្រូវយកលេខកូដចំនួន ៦ ខ្ទង់នោះមកបំពេញក្នុងប្រអប់ក្រហម ដូចរូបខាងក្រោមរួចចុចប៊ូតុងដែលមាន ពាក្យថា ផ្ទៀងផ្ទាត់ ជាការស្រេច។

ជំហានទី៥

  • បើសិនជាកម្មវិធី KOOMPI Hotspot ផ្ទៀងផ្ទាត់ថាលោកអ្នកបំពេញលេខកូដបាន ត្រឹមត្រូវហើយនេាះវានឹងចេញផ្ទាំងមួយទៀត ដូចរូបខាងក្រោមដើម្បីអោយអ្នកបំពេញ ទម្រង់ព័ត៍មានផ្ទាល់ខ្លួន។ លោកអ្នកត្រូវតែបំពេញ ឈ្មោះ ថ្ងៃ ខែ ឆ្នាំកំណើត ទីតាំង និង ភេទ រូចចុចសញ្ញាគ្រីសនៅខាង់លើជាការស្រេច។

ជំហានទី៦

  • នៅពេលដែលអ្នកបំពេញព័ត៍មាននៅ ជំហានទី ៥ រួចរាល់ហើយអ្នកនឹងចូលដល់ ផ្ទាងសូមស្វាគមន៍ដូចរូបខាងក្រោម។ បន្ទាប់មកអ្នកត្រូវបំពេញ លេខទូរស័ព្ធ និង ពាក្យសម្ងាត់ ដែលអ្នកបានបង្កើត នៅជំហានទី ៣ រួចចុចប៊ូតុង ចូលគណនី ជាការស្រច។

ជំហានទី៧

  • ក្រោយពីអ្នកចូលទៅគណនីបានត្រឹមត្រូវ នៅជំហានទី ៦ ហើយអ្នកនិងឃើញផ្ទាង KOOMPI Fi-Fi ដូចរូបខាងក្រោម។ បន្ទាប់មកអ្នកត្រូវបង្កើតកន្លែងផ្ទុកប្រាក់ ដោយចុចលើប៊ូតុង យកកាបូប។

  • បន្ទាប់ពីចុចលើប៊ូតុង យកកាបូប ហើយអ្នកនិងចូលដ់ល់ផ្ទាងមួយទៀត ដូចរូបខាងក្រោមទី ១ រូចអ្នកត្រូវចុច លើប៊ូតុង យកកាបូប ដើម្បីធ្វើការបង្កើត កាបូប រួចអ្នកនឹងឃើញសារមួយប្រាប់ថា អ្នកទទួលបានប្រាក់ចំនួន ១០០ SEL ដូចរូបទី ២ បានន័យថាឥឡូវអ្នកមាន ប្រាក់ ១០០ SEL ដើម្បីយកទៅទិញ គម្រោង KOOMPI Fi-Fi ។ បន្ទាប់ មកចុចប៊ូតុង បិទ។
